NAME

generate_gran_topology_files - creates the ENIQ topology directories and files for BSC nodes.

VERSION

This documentation refers to generate_gran_topology_files.pl version 1.0.

USAGE

No arguments required, just run using the command:

generate_gran_topology_files.pl

REQUIRED ARGUMENTS

None

OPTIONS

None

DESCRIPTION

This script creates all the topology information needed for the ENIQ GRAN nodes.

For the specification of the contents of the topology files, see ``Interwork Description for ENIQ-M xml files'', 1/15519-APR 901 0199.

Directories

The topology directories created are:

  • /eniq/data/pmdata/eniq_oss_1/gran
  • /eniq/data/pmdata/eniq_oss_1/gran/topologyData
  • /eniq/data/pmdata/eniq_oss_1/gran/topologyData/CELL
  • /eniq/data/pmdata/eniq_oss_1/gran/topologyData/DIM_BSS_AREA
  • /eniq/data/pmdata/eniq_oss_1/gran/topologyData/GranNetwork
  • /eniq/data/pmdata/eniq_oss_1/gran/topologyData/Site

CELL

The Cell topology files are stored in the CELL directory, an example file is:

  • /eniq/data/pmdata/eniq_oss_1/gran/topologyData/CELL/CELL

CELL SET

The Cell Set topology files are stored in the DIM_BSS_AREA directory, an example file is:

  • /eniq/data/pmdata/eniq_oss_1/gran/topologyData/DIM_BSS_AREA/DIM_BSS_AREA

BSC

The BSC topology files are stored in the BSC directory, an example file is:

  • /eniq/data/pmdata/eniq_oss_1/gran/topologyData/GranNetwork/BSC

Site

The Site topology files are stored in the Site directory, an example file is:

  • /eniq/data/pmdata/eniq_oss_1/gran/topologyData/Site/Site

NAME

generate_utran_topology_files - creates the ENIQ topology directories and files for RNC nodes.

VERSION

This documentation refers to generate_utran_topology_files.pl version 1.0.

USAGE

No arguments required, just run using the command:

generate_utran_topology_files.pl

REQUIRED ARGUMENTS

None

OPTIONS

None

DESCRIPTION

This script creates all the topology information needed for the ENIQ RNC nodes.

For the specification of the contents of the topology files, see ``Interwork Description for ENIQ-M xml files'', 1/15519-APR 901 0199.

Directories

The topology directories created are:

  • /eniq/data/pmdata/eniq_oss_1/utran
  • /eniq/data/pmdata/eniq_oss_1/utran/topologyData
  • /eniq/data/pmdata/eniq_oss_1/utran/topologyData/RNC
  • /eniq/data/pmdata/eniq_oss_1/utran/topologyData/Site

RNC

The RNC topology files are stored in the RNC directory, an example file is:

  • /eniq/data/pmdata/eniq_oss_1/utran/topologyData/RNC/SubNetwork_rnc01_MeContext_rnc01.xml

Site

The Site topology files are stored in the Site directory, an example file is:

  • /eniq/data/pmdata/eniq_oss_1/utran/topologyData/Site/sites.xml
